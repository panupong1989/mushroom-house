## RS485 bus
- Modbus RTU, บัสเดียว A/B/GND, ต่อพ่วง 3 ตัว (slave address ไม่ซ้ำ: 1,2,3)
- ใส่ termination 120Ω คร่อม A-B ที่ปลายทั้งสองข้าง
- ใช้ transceiver 3.3V (MAX3485/SP3485) หรือบอร์ด auto-direction; ถ้าใช้ manual DE/RE ผูก 1 GPIO
- เซนเซอร์แนะนำ: XY-MD02 / SHT20-RS485 (register: temp, humidity)

## 1-wire (DS18B20 ×3)
- ต่อพ่วงสายเดียว, pull-up 4.7kΩ, แต่ละตัวมี ROM address เฉพาะ
- วางหัว/กลาง/ท้ายโรง จิ้มในเบดลึก 5–10 ซม.

## ไฟ + เดินสาย
- ไฟ 220V อยู่ในตู้คอนโทรลนอกโรงทั้งหมด — เข้าโรงเฉพาะ 24V + สายสัญญาณ
- สายสัญญาณเดินแยกท่อจากสาย 220V/ฮีทเตอร์ (กัน noise)
- กล่อง IP65 ทั้งหมด

## ความปลอดภัย (ฮาร์ดแวร์ — ห้ามพึ่ง firmware อย่างเดียว)
- **Thermal cutoff ฮีทเตอร์:** thermostat/thermal fuse อนุกรม ตัดไฟเองไม่ขึ้นกับ ESP32
- **Float dry-run:** น้ำต่ำตัดปั๊มได้ระดับฮาร์ดแวร์
- **RCD/กันดูด** ทุกวงจร; relay **fail-safe** = ไฟดับ → ทุกโหลด OFF
- **Hardware watchdog** ให้ ESP32 รีบูตเองเมื่อค้าง
